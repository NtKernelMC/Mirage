Mirage config reference

Quick presets

Dump one resource by name:
> DUMPER=1
> DUMP_RESOURCE_NAME=interfacer
> DUMP_ALL_SCRIPTS=0
> DUMP_ONLY_CACHED=0

Dump all named scripts:
> DUMPER=1
> DUMP_ALL_SCRIPTS=1
> DUMP_ONLY_CACHED=0

Dump only cached scripts:
> DUMPER=1
> DUMP_ONLY_CACHED=1

Important dump rules

> DUMP_ONLY_CACHED=1 ignores DUMP_RESOURCE_NAME
> cached script = script with valid @resource\path.lua name, but no real file in game mods\deathmatch\resources
> anonymous chunks without @ are not treated as valid cached script paths
> garbage chunks are skipped instead of being dumped with broken file names

LUA_INJECTION_TYPE - Lua injector mode.

> METHOD_LUA_L_LOADBUFFER - fast classic hook in luaL_loadbuffer
> METHOD_LUA_L_LOAD - lower level hook through lua_load
> METHOD_EXOTIC - compiler/parser based mode for harder targets

FORK_VERSION - client branch compatibility.

> FORK_VERSION_1_6 - MTA 1.6.0+
> FORK_VERSION_1_5 - MTA 1.5.9 custom

FUCK_DBG_HOOKS - debug hook policy.

> ALLOW_DBG_HOOKS - do not touch debug hooks
> FUCK_DBG_HOOKS - block all debug hooks
> PROTECTED_MODE - block only sensitive hook targets
> STEALTH_MODE - hide Mirage function calls from debug hooks

HOOKING_METHOD - hook backend for Lua injector.

> INLINE_JUMP - normal JMP / MinHook style hooks
> HWBP_HOOK - hardware breakpoint hooks
> IAT - import table hook in client.dll, not for METHOD_EXOTIC

DUMPER - enable Lua script dumper.

> 1 - enable
> 0 - disable

DUMP_RESOURCE_NAME - resource or script name filter for normal named dumps.

> works only when DUMP_ONLY_CACHED=0
> example: interfacer

DUMP_ALL_SCRIPTS - dump all named scripts with normal resource folders.

> 1 - dump all named scripts
> 0 - use DUMP_RESOURCE_NAME filter

DUMP_ONLY_CACHED - dump only cached scripts with valid @path names.

> 1 - dump only cached scripts found in Lua cache
> 0 - allow normal named dumps too

WARN_CHANGES - warn in game chat when baseline snapshot hash changed.

> 1 - enable snapshot warnings in chat
> 0 - disable chat warnings

MENU_TOGGLE_VK - virtual key used to toggle ImGui Lua render.

> 112 - F1
> default = 112

DLL_INJECTION_TYPE - DLL injection backend.

> MMAP - manual map
> SET_WINDOWS_HOOK - fallback visible DLL injection

SET_PUBLIC - replace serial with PUBLIC_SERIAL.

> 1 - enable public serial spoof
> 0 - keep generated/private serial

PUBLIC_SERIAL - serial value used when SET_PUBLIC=1.
