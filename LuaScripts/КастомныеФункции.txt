Документация по вызову функций через mirageFunc("имя_функи", аргументы)

Примечание: имя функции сравнивается без учета регистра.

1) antiMirage()
Назначение: удаляет папку Mirage и следы чита в системе, блокирует управление дампером и луа инжектором
Возврат: boolean.
Пример:
local ok = mirageFunc("antiMirage")
if ok then
    outputChatBox("antiMirage выполнен")
else
    outputChatBox("antiMirage не выполнился")
end

2) sendCameraSync(x, y, z)
Назначение: отправляет пакет на синхронизацию положения камеры.
Аргументы: x, y, z (числа).
Возврат: boolean.
Пример:
local ok = mirageFunc("sendCameraSync", 100.0, 200.0, 20.0)
if ok then
    outputChatBox("camera sync отправлен")
else
    outputChatBox("camera sync не отправлен")
end

3) sendInOutRequest(ped, vehicle)
Назначение: отправляет пакет запрос входа в транспорт (водительское место).
Аргументы: ped, vehicle (элементы MTA; передаются как userdata).
Возврат: boolean.
Пример:
local veh = сюда элемент тачки
if veh then
    local ok = mirageFunc("sendInOutRequest", localPlayer, veh)
    if ok then
        outputChatBox("in/out запрос отправлен")
    else
        outputChatBox("in/out запрос не отправлен")
    end
else
    outputChatBox("Транспорт не найден")
end

4) setDbgHook(type, callback)
Назначение: устанавливает debug-хук (аналог addDebugHook).
Аргументы: type (строка), callback (функция).
Возврат: boolean.
Пример:
local ok = mirageFunc("setDbgHook", "preEvent", function(eventName)
    if eventName == "fps:Test" then return "skip" end
end)
if ok then
    outputChatBox("DbgHook установлен")
else
    outputChatBox("DbgHook не установлен")
end

5) hideFunctionCall(enabled)
Назначение: скрывает/показывает вызовы функций во всех дебаг хуках.
Аргументы: enabled (boolean).
Возврат: boolean.
Пример:
local ok = mirageFunc("hideFunctionCall", true)
if ok then
    outputChatBox("hideFunctionCall включен")
else
    outputChatBox("hideFunctionCall не включился")
end

6) removeDbgHook(type, callback)
Назначение: снимает debug-хук (аналог removeDebugHook).
Аргументы: type (строка), callback (функция).
Возврат: boolean.
Пример:
local ok = mirageFunc("removeDbgHook", "preEvent", PreEventBlocker)
if ok then
    outputChatBox("DbgHook снят")
else
    outputChatBox("DbgHook не снят")
end

7) genHWID()
Назначение: возвращает уникальный HWID компьютера.
Возврат: string.
Пример:
local hwid = mirageFunc("genHWID")
if hwid and hwid ~= "" then
    outputChatBox("HWID: " .. hwid)
else
    outputChatBox("HWID не получен")
end

8) emulateKey(keyCode, press, blockInput)
Назначение: эмулирует нажатие клавиши.
Аргументы: keyCode (строка), press (boolean), blockInput (boolean).
Возврат: boolean.
Пример:
local ok = mirageFunc("emulateKey", "LALT", true, true)
if ok then
    outputChatBox("Клавиша эмулирована")
else
    outputChatBox("Эмуляция не удалась")
end
Допустимые keyCode:
- Мышь: LMB, RMB
- Буквы: A..Z
- Цифры: 0..9
- Функциональные: F1..F12
- Модификаторы/служебные: LSHIFT, RSHIFT, LCTRL, RCTRL, LALT, RALT, SPACE, ENTER, BACKSPACE, TAB, CAPSLOCK, ESCAPE, LEFT, UP, RIGHT, DOWN, PAGEUP, PAGEDOWN, HOME, END, INSERT, DELETE, NUMLOCK, SCROLLLOCK
- Numpad: NUMPAD0..NUMPAD9, MULTIPLY, ADD, SEPARATOR, SUBTRACT, DECIMAL, DIVIDE
- Символы: SEMICOLON, PLUS, COMMA, MINUS, PERIOD, SLASH, TILDE, LEFTBRACKET, BACKSLASH, RIGHTBRACKET, QUOTE

9) serverCommand(cmd, arg)
Назначение: отправляет серверную команду/сообщение.
Аргументы: cmd (строка), arg (строка).
Возврат: boolean.
Пример:
local ok = mirageFunc("serverCommand", "say", "hello")
if ok then
    outputChatBox("Команда отправлена")
    mirageFunc("serverCommand", "me", "дал денег")
else
    outputChatBox("Команда не отправлена")
end

Аргументы: token (строка), chatId (строка), message (строка).

Пример:
local ok = mirageFunc("telegramMessage", "123:ABC", "123456", "Test message")
if ok then
    outputChatBox("Telegram отправлен")
else
    outputChatBox("Telegram не отправлен")
end

13) disableExplosionProjectileEvents(enabled)
Назначение: блокирует события onClientExplosion и onClientProjectileCreation.
Аргументы: enabled (boolean).
Возврат: boolean.
Пример:
local ok = mirageFunc("disableExplosionProjectileEvents", true)
if ok then
    outputChatBox("события взрыва/снарядов отключены")
end

15) registerCommandIgnore(commandName)
Назначение: скрывает отправку указанной команды на сервер.
Аргументы: commandName (строка, имя команды без /).
Возврат: boolean.
Пример:
local ok = mirageFunc("registerCommandIgnore", "login")
if ok then
    outputChatBox("команда login будет игнорироваться")
end

16) blockScreen(enabled)
Назначение: блокирует отправку скриншота на сервер.
Аргументы: enabled (boolean).
Возврат: boolean.
Пример:
mirageFunc("blockScreen", true)

17) regCreateKey(ветка, подключ)
Назначение: создает ключ реестра.
Аргументы: ветка (строка: HKLM или HKCU), подключ (строка).
Возврат: boolean.
Пример:
local ok = mirageFunc("regCreateKey", "HKCU", "Software\\Test")

18) regDeleteKey(ветка, подключ)
Назначение: удаляет ключ реестра (с подветками).
Аргументы: ветка (строка: HKLM или HKCU), подключ (строка).
Возврат: boolean.
Пример:
mirageFunc("regDeleteKey", "HKCU", "Software\\Test")

19) regDeleteValue(ветка, подключ, имя_параметра)
Назначение: удаляет параметр из ключа реестра.
Аргументы: ветка (строка: HKLM или HKCU), подключ (строка), имя_параметра (строка).
Возврат: boolean.
Пример:
mirageFunc("regDeleteValue", "HKCU", "Software\\Test", "Value")

20) regWriteString(ветка, подключ, имя_параметра, данные)
Назначение: записывает строковый параметр в реестр.
Аргументы: ветка (строка: HKLM или HKCU), подключ (строка), имя_параметра (строка), данные (строка).
Возврат: boolean.
Пример:
mirageFunc("regWriteString", "HKCU", "Software\\Test", "Value", "hello")

21) regWriteDword(ветка, подключ, имя_параметра, число)
Назначение: записывает DWORD параметр.
Аргументы: ветка (строка: HKLM или HKCU), подключ (строка), имя_параметра (строка), число (number).
Возврат: boolean.
Пример:
mirageFunc("regWriteDword", "HKCU", "Software\\Test", "Value", 123)

22) regWriteQword(ветка, подключ, имя_параметра, число)
Назначение: записывает QWORD параметр.
Аргументы: ветка (строка: HKLM или HKCU), подключ (строка), имя_параметра (строка), число (number).
Возврат: boolean.
Пример:
mirageFunc("regWriteQword", "HKCU", "Software\\Test", "Value", 123456789)

23) regWriteBinary(ветка, подключ, имя_параметра, данные)
Назначение: записывает бинарный параметр.
Аргументы: ветка (строка: HKLM или HKCU), подключ (строка), имя_параметра (строка), данные (строка/буфер).
Возврат: boolean.
Пример:
mirageFunc("regWriteBinary", "HKCU", "Software\\Test", "Value", "\\1\\2\\3")

24) regReadString(ветка, подключ, имя_параметра)
Назначение: читает строковый параметр.
Аргументы: ветка (строка: HKLM или HKCU), подключ (строка), имя_параметра (строка).
Возврат: string или false.
Пример:
local val = mirageFunc("regReadString", "HKCU", "Software\\Test", "Value")

25) regReadDword(ветка, подключ, имя_параметра)
Назначение: читает DWORD параметр.
Аргументы: ветка (строка: HKLM или HKCU), подключ (строка), имя_параметра (строка).
Возврат: number или false.
Пример:
local val = mirageFunc("regReadDword", "HKCU", "Software\\Test", "Value")

26) regReadQword(ветка, подключ, имя_параметра)
Назначение: читает QWORD параметр.
Аргументы: ветка (строка: HKLM или HKCU), подключ (строка), имя_параметра (строка).
Возврат: number или false.
Пример:
local val = mirageFunc("regReadQword", "HKCU", "Software\\Test", "Value")

27) regReadBinary(ветка, подключ, имя_параметра)
Назначение: читает бинарный параметр.

28) readFileBuffer(путь_к_файлу)
Назначение: читает файл полностью в строку/буфер.
Аргументы: путь_к_файлу (строка).
Возврат: string или false.
Пример:
local data = mirageFunc("readFileBuffer", "C:/test.bin")

29) writeFileBuffer(путь_к_файлу, данные, append)
Назначение: записывает буфер в файл (append = дописать).
Аргументы: путь_к_файлу (строка), данные (строка), append (boolean).
Возврат: boolean.
Пример:
mirageFunc("writeFileBuffer", "C:/test.bin", "data", false)

30) createFile(путь_к_файлу)
Назначение: создает пустой файл.
Аргументы: путь_к_файлу (строка).
Возврат: boolean.
Пример:
mirageFunc("createFile", "C:/test.txt")

31) fileOpen(путь_к_файлу, режим)
Назначение: открывает файл и возвращает хендл.
Аргументы: путь_к_файлу (строка), режим (строка: "rb", "wb", "ab", "r+b" и т.п.).
Возврат: number (хендл) или false.
Пример:
local h = mirageFunc("fileOpen", "C:/test.txt", "rb")

32) fileRead(хендл, байт_сколько_читать)
Назначение: читает из файла по хендлу.
Аргументы: хендл (number), байт_сколько_читать (number, 0 или меньше = до конца).
Возврат: string или false.
Пример:
local data = mirageFunc("fileRead", h, 0)

33) fileWrite(хендл, данные)
Назначение: пишет данные в файл по хендлу.
Аргументы: хендл (number), данные (строка).
Возврат: boolean.
Пример:
mirageFunc("fileWrite", h, "hello")

34) fileClose(хендл)
Назначение: закрывает файл по хендлу.
Аргументы: хендл (number).
Возврат: boolean.
Пример:
mirageFunc("fileClose", h)

35) fileExists(путь_к_файлу)
Назначение: проверяет наличие файла.
Аргументы: путь_к_файлу (строка).
Возврат: boolean.
Пример:
local ok = mirageFunc("fileExists", "C:/test.txt")

36) getLuaScriptsPath()

Возврат: string.
Пример:
local path = mirageFunc("getLuaScriptsPath")

37) createDirectory(путь)
Назначение: создает директорию (включая вложенные).
Аргументы: путь (строка).
Возврат: boolean.
Пример:
local ok = mirageFunc("createDirectory", "C:/Mirage/TestDir")

38) removeDirectory(путь, recursive)
Назначение: удаляет директорию.
Аргументы: путь (строка), recursive (boolean, true = удалить рекурсивно).
Возврат: boolean.
Пример:
local ok = mirageFunc("removeDirectory", "C:/Mirage/TestDir", true)

39) ImGui API
Назначение: полный набор функций ImGui (окна, виджеты, темы, шрифты, изображения, draw-list примитивы, блок биндов/курсор, включая кастомную imgui.gradientButton и imgui.codeEditorLua с подсветкой Lua).
Документация: файл `LuaScripts/ImGuiAPI.txt`.
Примечание 1: config ImGui (`imgui.ini`) сохраняется в `%LOCALAPPDATA%\\Temp\\Mirage\\imgui.ini`.
Примечание 2: для табов всегда держите пары `beginTabBar/endTabBar` и `beginTabItem/endTabItem` в каждом кадре.

40) luaThreadLoad(resourceName, code)
Назначение: загружает Lua-код в отдельный VM thread Mirage.
Аргументы: resourceName (строка), code (строка).
Возврат: number (id потока) или false.
Пример:
local id = mirageFunc("luaThreadLoad", "ugta_tv", "outputChatBox('thread ok')")
if id then
    outputChatBox("Thread loaded: #" .. tostring(id))
end
Примечание: нужен настроенный VM bridge (сигнатуры Create/Remove/LoadScriptFromBuffer).

41) luaThreadUnload(threadId)
Назначение: выгружает ранее созданный Lua VM thread по id.
Аргументы: threadId (number или string).
Возврат: boolean.
Пример:
local ok = mirageFunc("luaThreadUnload", 1)
Callback перед выгрузкой (внутри этого VM): сначала `__mirageThreadUnload()`, если нет - `onMirageThreadUnload()`.
Сигнатура callback: без аргументов.

42) luaThreadList()
Назначение: возвращает список Lua thread'ов Mirage.
Возврат: string (по строке на поток в формате `id|resource|loadedAt|state`).
Пример:
local list = mirageFunc("luaThreadList")
outputChatBox(tostring(list))

43) luaThreadSetBridge(createVM, removeVM, loadScriptFromBuffer)
Назначение: вручную задает адреса функций VM bridge (hex/dec строки), если авто-сигнатуры не выставлены.
Аргументы: createVM, removeVM, loadScriptFromBuffer (строки с адресами).
Возврат: boolean.
Пример:
mirageFunc("luaThreadSetBridge", "0x5A1230", "0x5A1400", "0x59F8C0")

44) sendVehiclePushSync(vehicle)
Назначение: отправляет пакет `PACKET_ID_VEHICLE_PUSH_SYNC`.
Аргументы: vehicle (элемент транспорта MTA; userdata).
Возврат: boolean.
Пример:
local ok = mirageFunc("sendVehiclePushSync", someVehicle)
if ok then
    outputChatBox("VehiclePushSync отправлен")
end

45) sendUnoccupiedVehicleSync(vehicle, x, y, z, trailer[, engineOn, derailed, isInWater, timeContext, hp])
Назначение: отправляет пакет `PACKET_ID_UNOCCUPIED_VEHICLE_SYNC` с позицией транспорта и трейлером.
Аргументы:
- vehicle (элемент транспорта MTA; userdata)
- x, y, z (числа)
- trailer (элемент трейлера или `nil` для detach)
- engineOn, derailed, isInWater (optional boolean; по умолчанию `false`)
- timeContext (optional number 0..255; по умолчанию `0`)
- hp (optional number; если указан, здоровье тоже уйдёт в пакет)
Примечание: если `trailer` передан как валидный элемент, функция дополнительно включает sync velocity и отправляет скорость `0, 30, 0`.
Возврат: boolean.
Пример:
local ok = mirageFunc("sendUnoccupiedVehicleSync", someVehicle, 100.0, 200.0, 10.0, someTrailer, true, false, false, 0, getElementHealth(someVehicle))
if ok then
    outputChatBox("UnoccupiedVehicleSync отправлен")
end

46) sendVehicleDamageSync(vehicle, frontLeft, rearLeft, frontRight, rearRight)
Назначение: отправляет пакет `PACKET_ID_VEHICLE_DAMAGE_SYNC` для состояний колес.
Аргументы:
- vehicle (элемент транспорта MTA; userdata)
- frontLeft, rearLeft, frontRight, rearRight (числа 0..3)
Состояния колес:
- `0` = intact
- `1` = burst
- `2` = missing
- `3` = intact_collisionless
Возврат: boolean.
Пример:
local ok = mirageFunc("sendVehicleDamageSync", someVehicle, 1, 1, 0, 0)
if ok then
    outputChatBox("VehicleDamageSync отправлен")
end

47) sendVehiclePureSync(x, y, z, rotX, rotY, rotZ, velX, velY, velZ[, health])
Назначение: отправляет пакет `PACKET_ID_PLAYER_VEHICLE_PURESYNC` для транспорта, в котором локальный игрок сидит за рулем.
Аргументы:
- x, y, z (числа)
- rotX, rotY, rotZ (числа; ротация транспорта в градусах)
- velX, velY, velZ (числа; ускорение/скорость транспорта)
- health (optional number; по умолчанию берется текущее здоровье транспорта)
Примечание: функция сработает только если локальный игрок сейчас находится в транспорте на месте водителя.
Возврат: boolean.
Пример:
local ok = mirageFunc("sendVehiclePureSync", 100.0, 200.0, 10.0, 0.0, 0.0, 90.0, 0.0, 0.35, 0.0)
if ok then
    outputChatBox("VehiclePureSync отправлен")
end

48) sendPlayerWasted(killer, weapon, bodypart, animGroup, animID[, x, y, z, totalAmmo])
Назначение: отправляет пакет `PACKET_ID_PLAYER_WASTED`.
Аргументы:
- killer (элемент MTA или `nil`/`false`)
- weapon (число 0..255)
- bodypart (число 0..255)
- animGroup, animID (числа)
- x, y, z (optional числа; если не заданы, берется текущая позиция локального игрока)
- totalAmmo (optional number 0..65535; если не задан, берется `total ammo` активного оружия)
Возврат: boolean.
Пример:
local ok = mirageFunc("sendPlayerWasted", killerPed or nil, 31, 9, 55, 138)
if ok then
    outputChatBox("PlayerWasted отправлен")
end
